# GoogleTransit-iOS6

Forked from https://github.com/simonmaddox/GoogleTransit-iOS6

修改為跳轉中文版 Google Maps

iOS 6 地圖 App 沒有大眾運輸規劃功能。

安裝此程式即可在 iOS 6 地圖 App 規劃大眾運輸路線，自動跳轉到 Google Maps mobile web 版。
